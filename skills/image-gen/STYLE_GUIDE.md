# STYLE GUIDE: Neko Teacher Illustrations

## Universal Style Prefix

Every prompt must begin with this base template:

> A cute chibi anime catgirl teacher with pink hair, cat ears, glasses, wearing a white lab coat with pink bow, explaining technical concepts in a friendly way, soft hand-drawn illustration style mixed with professional academic diagram, clear labels and arrows, pastel pink-purple-blue color palette, kawaii 2D anime style with light watercolor and marker texture, clean educational layout, high clarity, professional yet adorable, white background with subtle sparkles and cherry blossom elements

## Composition Rules

1. **Concept First**: The diagram, formula, or architecture takes 60-70% of the frame. Neko occupies the side or corner, pointing at it.

2. **Consistent Palette**: Pastel pink, soft purple, light blue. Warm but not overwhelming.

3. **Labels and Arrows**: All technical elements must have readable labels. Mix Chinese and English where appropriate.

4. **White Background**: Clean, minimal. Subtle sparkles or cherry blossom petals are acceptable but not distracting.

5. **No Style Drift**: The same hand-drawn/watercolor texture across all images. No photorealistic rendering. No 3D effects.

## Technical Accuracy

- Formulas must be mathematically correct.
- Architecture diagrams must match the actual model structure.
- Tensor shapes and dimensions should be labeled when relevant.
- Arrows must indicate real data flow, not decorative direction.

## Output Spec

- Size: 1024x1024
- Format: PNG
- Transparent background: No (white background preferred for PDF compatibility)
